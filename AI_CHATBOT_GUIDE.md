## 🚀 How to Use

### Step 1: Upload Your Data
```
1. Go to "Upload Data" page
2. Upload a CSV file
3. Wait for processing
4. Data is stored in localStorage
```

### Step 2: Open AI Chatbot
```
1. Click "AI Chatbot" in sidebar
2. You'll see a welcome message
3. The chatbot will confirm your dataset is loaded
4. Start asking questions!
```

### Step 3: Ask Questions
```
Type your question in the input field or click quick question buttons
Examples:
• "Summarize my dataset"
• "Show average values"
• "Detect anomalies"
• "Analyze trends"
```

## 🔧 Technical Details

### How It Works

1. **Data Storage**
   ```javascript
   // Data is stored in localStorage
   localStorage.setItem('dataset', JSON.stringify(datasetObject))
   ```

2. **Natural Language Processing**
   ```javascript
   // The chatbot uses keyword matching and pattern recognition
   if (question.includes('summarize') || question.includes('summary')) {
     // Generate summary response
   }
   ```

3. **Statistical Calculations**
   ```javascript
   // Real statistical analysis on your data
   const avg = values.reduce((a, b) => a + b, 0) / values.length;
   const median = sortedValues[Math.floor(values.length / 2)];
   const stdDev = calculateStandardDeviation(values);
   ```

4. **Anomaly Detection**
   ```javascript
   // Uses 2 standard deviations from mean
   const anomalies = values.filter(val => 
     Math.abs(val - mean) > 2 * standardDeviation
   );
   ```

## 🆘 Troubleshooting

### Issue: "No Dataset Found"
**Solution:**
1. Go to "Upload Data" page
2. Upload a CSV file
3. Return to AI Chatbot
4. Refresh if needed

### Issue: No Response to Query
**Solution:**
- Check if dataset is loaded
- Try rephrasing your question
- Use quick question buttons
- Ask "help" to see capabilities

### Issue: Incorrect Analysis
**Solution:**
- Verify your CSV data format
- Check for special characters
- Ensure numeric columns are properly formatted
- Try uploading the file again
